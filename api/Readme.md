# API

Para conseguir utilizar a API (JsonSERVER) tive que juntar todos os arquivos em um arquivo só.
O arquivo está nomeado como "geral.json".

Para conseguir rodar, é necessário que você tenha o pacote Json-server instalado em sua máquina.

https://www.npmjs.com/package/json-server

```bash
#install json-server
$npm i json-server

#run the server
$json-server -p=3333 --watch geral.json

```
